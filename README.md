# AIOps XP

Repositório de apoio para o evento **AIOps Xperience** — um curso prático de 3 dias focado em Inteligência Artificial aplicada a DevOps, Cloud e SRE.

## Sobre o Evento

O AIOps Xperience é um evento hands-on onde os participantes constroem, do zero, um agente AIOps funcional capaz de monitorar eventos de Kubernetes, identificar causa raiz automaticamente e notificar via Discord com um plano de ação.

## Estrutura do Evento

| Dia | Tema | Objetivo |
|-----|------|----------|
| **01** | Fundamentos de IA e Engenharia de Prompt | Base conceitual em IA, ML e técnicas de prompt aplicadas a DevOps |
| **02** | Codificação Agêntica e MCP | Uso prático do Gemini CLI e criação de MCP Server para Kubernetes |
| **03** | Agente AIOps com N8n | Construção do agente AIOps funcional end-to-end |

## Stack Tecnológica

- **Google Gemini 2.5 Pro** — Modelo de linguagem (gratuito via CLI)
- **Gemini CLI** — Codificação agêntica no terminal
- **Model Context Protocol (MCP)** — Padrão para conectar IA a sistemas externos
- **N8n** — Orquestração de workflows com IA
- **Kubernetes** — Alvo de monitoramento e diagnóstico
- **Docker / Docker Compose** — Setup de ambiente local
- **Discord** — Canal de notificação do agente

## Pré-requisitos

- Docker (criação e uso de imagens)
- Docker Compose (criação de stacks)
- Kubernetes básico (criar deployments)
